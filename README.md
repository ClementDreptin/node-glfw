# node-glfw
GLFW bindings for Node.js

## Installation

### Requirements
Make sure you have `libglfw3` and `libglfw3-dev` installed on your machine. If not, run this:
```
sudo apt-get install libglfw3 libglfw3-dev
```

### Generates the Makefiles and compile
```
npm run build
```

### Run the example
```
node index
```

## Limitations
Only **Linux** building is supported at the moment.
